FreeCodeCamp API Basejump: URL Shortener Microservice

User stories:

  • I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • When I visit that shortened URL, it will redirect me to my original link.

Example Usages:

https://api-developments-ruja.c9users.io/urlshortener/https://www.google.com

https://api-developments-ruja.c9users.io/urlshortener/http://foo.com:80

Example Output:

{ "original_url":"http://foo.com:80", "short_url":"https://api-developments-ruja.c9users.io/urlshortener/8170" }

Usage:

https://api-developments-ruja.c9users.io/urlshortener/8170

Will redirect to:

http://foo.com:80
